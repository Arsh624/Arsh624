### 🌱 Recent Projects
- 🤖 **[Regime-Switching Multi-Agent RL for Portfolio Optimization](#)** – multi-agent reinforcement learning system with PPO agents adapting to different market regimes.  
- 🧠 **[RepoGraph V2 – Distributed Debugging Platform](#)** – graph-based system that analyzes dependency graphs to localize flaky tests across large codebases.  
- ✈️ **[HackCU AI Travel Planner](#)** – multi-agent trip planning system that aggregates group preferences and generates optimized itineraries using LLM workflows.  
- ⚡ **[Real-Time Anomaly Detection Service](#)** – streaming ML pipeline processing event data and exposing anomaly predictions through backend APIs.  
- 📝 **[LoanSphere – AI Loan Approval Prediction](https://github.com/Arsh624/LoanSphere)** – ML-based decision support system for loan approval prediction.
